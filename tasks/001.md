 # Task 001: EdgeOne 根目录软链接

 **状态**: ✅ 已完成

 ## 目标

 在根目录创建软链接，使 EdgeOne Pages 部署时无需手动输入 `packages/edgeone/`。

 ## 创建的软链接

 | 根目录文件 | 指向 |
 |------------|------|
 | `edgeone.json` | `packages/edgeone/edgeone.json` |
 | `edgeone.config.json` | `packages/edgeone/edgeone.config.json` |
 | `.edgeoneignore` | `packages/edgeone/.edgeoneignore` |
 | `.env.example` | `packages/edgeone/.env.example` |
 | `build.mjs` | `packages/edgeone/build.mjs` |
 | `package.json` | `packages/edgeone/package.json` |
 | `edge-functions/` | `packages/edgeone/edge-functions/` |
 | `scripts/` | `packages/edgeone/scripts/` |

 ## 验证

 - `node build.mjs` 从根目录执行成功 ✅
 - `edge-functions/` 目录通过软链接正确解析 ✅
 - EdgeOne Pages 现在可以将 Root Directory 设为 `/` ✅
